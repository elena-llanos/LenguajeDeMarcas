Ejercicios para entregar

Lenguaje de Marcas 2021/2022

01_texto_estructurado.html
Escribe una pequeña página en la que tengas: cabecera h1, título, texto bien organizado con etiquetas. Usa alguna entidad tipo © (&copy;) Puedes usar los formatos especiales de los ejercicios 6 a 8. La página debe pasar el validador y estar en el servidor el viernes 15
02_texto_maquetado.html
Escribe algo similar al primer ejericio (puedes reutilizarlo si quieres ) y añade un fichero externo con reglas CSS para maquetar texto y ajustar las cajas con padding, border.... Usa 02_texto_maquetado.css como nombre del fichero de estilo.
03_horario.html
Escribe un documento con tu horario de clase de este curso. La tabla presentará una cabecera horizontal con los días de la semana y una cabecera en la primera columna con intervalors horarios del tipo 15:00-16:00. En el intervalo 18:00-18:30 aparecerá una celda con la palabra recreo que ocupe todo el ancho de tabla. Los módulos aparecerán con su nombre completo y si son sesiones de más de una hora, la celda correspondiente ocupará varias filas. Utiliza clases CSS para marcar cada módulo de su color correspondiente.

05_calendario.html
Para este ejercicio, bájate el calendario para este ejercicio. Añade clases o identificadores a determinadas celdas y proporciona un css adecuado. Por ejemplo, partiendo de la guía de centro puedes hacer clases para inicio/fin de evaluación, días de exámenes...
