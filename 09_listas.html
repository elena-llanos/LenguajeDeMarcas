Listas en HTML

En HTML tenemos tres tipos de listas

Listas no ordenadas

Las listas no ordenadas las realizaremos con la etiqueta <ol>

ul → unordered list

li →list items

  • Primer elemento
  • Segundo
  • Tercero

No es recomendable pero podemos usar el atributo type, para eso está el CSS para cambiar el formato


<ul type="disc"> <!-- "square" "circ" "disc"-->
<li>Primer elemento</li>
<li>Segundo</li>
<li>Tercero</li>
</ul>
