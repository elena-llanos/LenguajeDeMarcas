Gradientes en CSS

Cada vez van apareciendo más técnicas en CSS y HTML que realizan tares que antes iban con otras tecnologías

Por ejemplo, antes los fromularios se validaban con javascript y ahora lo hace HTML5

Como vamos a ver ahora, hay una forma de rellenar fondos llamada gradientes que antes se hacía con un programa de dibujo tipo photoshop

Un gradiente se consigue con una propiedad del tipo background-image: linear-gradient(direction, color-stop1, color-stop2, ...);

A continuación tienes algunos ejemplos Hay alguno más comentado en el fichero CSS

background-image: linear-gradient(purple, gold);

background-image: linear-gradient(90deg,purple, gold);

background-image: linear-gradient(to left, purple, gold);

background-image: linear-gradient(-135deg,purple, gold, navy);

background-image: linear-gradient(to bottom left, purple, gold, navy);

background-image: linear-gradient( to right, rgba(200,200,200,1), rgba(200,200,200,0) );

radial-gradient(50px, purple, gold);

radial-gradient(50px 100px, purple, gold);

radial-gradient(50% 25%, purple, gold)
